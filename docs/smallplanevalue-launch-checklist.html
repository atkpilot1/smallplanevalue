2. Namecheap email options (domain already there)

Option A — Free Email Forwarding (start here)

Included with domains on Namecheap DNS (BasicDNS / PremiumDNS / FreeDNS).

  1. Log in at namecheap.com → Domain List → Manage (smallplanevalue.com)
  2. Redirect Email → Add alias (e.g. hello) → forward to your main Gmail
  3. Advanced DNS → Mail Settings → select Email Forwarding (MX records set automatically)
  4. Test from a different email address (forwarding tests from the same destination often fail)

Note: Free forwarding is virtual — you receive at your domain address; it is not a full send/receive mailbox.

Docs: namecheap.com → Support → “How to set up Free Email Forwarding”

Option B — Private Email (upgrade later)

  • Full mailbox at hello@smallplanevalue.com
  • Send and receive; optional auto-forward to Gmail in webmail settings
  • Choose this when you reply to pilots regularly as the brand
Quick pick for beta: Set up only hello@smallplanevalue.com first. Add feedback@ later if you want a separate public address on the site — both can forward to the same Gmail.

3. Suggested alias names — which to create?

All of these are free forwarders on Namecheap (alias → your Gmail). You can add up to 100 on BasicDNS.

Alias Use on Recommendation
hello@ Facebook Page contact, footer, general “reach us” Create this one first. Friendly, normal for a small product.
feedback@ Site footer, Feedback tab “email us too”, forum posts Optional. Signals “tell us what’s wrong.” Same Gmail is fine.
contact@ Formal listings, press, partners Skip for now — redundant with hello@.
info@ Generic business contact Skip — feels corporate for a beta side project.
support@ Paid product / ticket system later Wait until you charge and need support workflow.

Minimum setup: one alias (hello). Nice-to-have: hello + feedback, both → same Gmail, two Gmail filters (or one filter for the whole domain).

4. Gmail filter setup (step-by-step)

Goal: mail forwarded from Namecheap lands in your main Gmail but is easy to find — labeled SmallPlaneValue, not buried in personal mail.

Filter A — catch everything from your domain (recommended)

  1. Open Gmail in a browser → click the search box → click the filter icon (sliders) on the right.
  2. In To, enter: @smallplanevalue.com
    This matches any forwarded alias (hello@, feedback@, etc.).
  3. Click Create filter (bottom right).
  4. Check: Apply the label → New label → name it SmallPlaneValue.
  5. Optional but useful: check Star it during beta so you notice pilot mail quickly.
  6. Do not check “Skip the Inbox” during beta — you want these visible.
  7. Click Create filter.

Filter B — separate feedback@ (only if you created that alias)

  1. New filter → To: feedback@smallplanevalue.com
  2. Apply label: SmallPlaneValue/Feedback (nested label under SmallPlaneValue)
  3. Star it (optional)

Filter C — catch replies when you post on BeechTalk / FB

People may reply to your personal email instead of hello@. Optional filter:

  1. Has the words: smallplanevalue OR "small plane value"
  2. Label: SmallPlaneValue (same label as above)

Gmail sidebar & phone

  • In Gmail left sidebar, click the label SmallPlaneValue to see only project mail.
  • On iPhone/Android Gmail app: the label appears under Labels; you can enable notifications for that label only (Settings → your account → Labels → SmallPlaneValue → Notify).
Replying as hello@smallplanevalue.com
Free Namecheap forwarding is receive-only. Replies will come from your personal Gmail address unless you upgrade to Namecheap Private Email or configure Gmail “Send mail as” with SMTP. For beta, replying from personal Gmail with a signature line (“SmallPlaneValue.com”) is fine.
